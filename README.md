# Bayou

Bayou is a full-stack platform powering the Bayou Exotics customer experience and administration tools.

## Applications

```text
client-prod/
```

Customer-facing SvelteKit application.

Features:

- Weekly menu
- Product pages
- Shopping cart
- Meet scheduling
- Availability calendar
- Featured deals

```text
admin/
```

Content management application.

Planned features:

- Manage menu
- Products
- Categories
- Featured deals
- Weekly drops
- Availability
- Meet cancellations
- Raffle
- Homepage content

```text
shared/
```

Shared TypeScript models used by both applications.

## Technology

- SvelteKit
- React
- TypeScript
- Tailwind CSS
- AWS

## Goals

- Content-driven architecture
- Shared domain models
- Reusable UI components
- Strong typing across applications
- Static customer site backed by CMS-managed content

## Status

Current focus:
