# Bayou Client

Customer-facing web application for **Bayou Exotics**.

Built with:

- SvelteKit
- TypeScript
- Tailwind CSS
- date-fns

## Features

- Weekly menu browsing
- Product detail pages
- Shopping cart
- Meet time selection
- Availability calendar
- Weekly raffle
- Featured deals
- Responsive design

## Project Structure

```text
src/
  lib/
    components/
    builders/
    stores/
    helpers/

  routes/
    +layout
    +page
    menu/
    cart/

shared/
  types/
```

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Content

The client consumes content generated by the Bayou CMS.

Examples include:

- Home
- Menu
- Root
- Availability

The application is intentionally built as a content-driven frontend with reusable UI components and builder/view-model helpers.
